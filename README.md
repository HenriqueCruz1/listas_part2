# 📋 Listas Part 2 — Exercícios em Python

Exercícios práticos sobre listas compostas e manipulação de dados em Python.

## 📚 Exercícios

**1. Cadastro de pessoas por peso**
Lê nome e peso de várias pessoas e exibe quantas foram cadastradas, além das mais pesadas e mais leves.

**2. Separador de pares e ímpares**
Recebe 7 valores e os separa em pares e ímpares dentro de uma lista única, exibindo ambos em ordem crescente.

**3. Matriz 3x3**
Preenche uma matriz 3x3 com valores digitados pelo usuário e a exibe formatada.

**4. Matriz 3x3 com análise**
Aprimora o exercício anterior calculando a soma dos valores pares, soma da terceira coluna e maior valor da segunda linha.

**5. Gerador de palpites para Mega Sena**
Gera jogos aleatórios com 6 números entre 1 e 60, sem repetição, organizados em ordem crescente.

**6. Boletim escolar**
Cadastra nome e duas notas de alunos, calcula a média e permite consultar as notas individualmente.

## 🛠 Tecnologias
- Python 3

```bash
python nome_do_arquivo.py
```
